# Feature Requiremtns - Blog Website

## Overview

A statically generated website presenting the author's written content: posts, book notes, and snippets. Optimised for reading experience, performance, and search discoverability.

| ID      | Requirement                                                                                                 | Priority    |
| ------- | ----------------------------------------------------------------------------------------------------------- | ----------- |
| BLOG-01 | The blog MUST be fully statically generated with no runtime server                                          | Must Have   |
| BLOG-02 | The blog MUST render pages for all content types: posts, book-notes, and snippets                           | Must Have   |
| BLOG-03 | Each post page MUST include the quiz widget component                                                       | Must Have   |
| BLOG-04 | The quiz widget MUST only render on post pages that have at least one associated question                   | Should Have |
| BLOG-05 | The blog MUST support MDX rendering: rich text, code blocks with syntax highlighting, and custom components | Must Have   |
| BLOG-06 | The blog MUST achieve a Lighthouse Performance score ≥ 95 and SEO score ≥ 95                                | Must Have   |
| BLOG-07 | The blog SHOULD include listing pages for each content type with pagination or infinite scroll              | Should Have |
| BLOG-08 | Listing pages SHOULD support filtering by tag                                                               | Should Have |
