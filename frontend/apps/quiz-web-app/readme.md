# Quiz Web App

Standalone CSR React app for spaced-repetition flashcard study.

## Stack

- React 19 + Vite (strict CSR — no SSR)
- TanStack Router (file-based routing)
- Zustand (state — Phase 3+)
- Tailwind CSS 4 + shadcn/ui components
- Static quiz data from `@prj--personal-portfolio--v3/shared--quiz-export`

## Commands

```bash
# from monorepo root
pnpm --filter @prj--personal-portfolio--v3/frontend--quiz-web-app dev
pnpm --filter @prj--personal-portfolio--v3/frontend--quiz-web-app build
pnpm --filter @prj--personal-portfolio--v3/frontend--quiz-web-app typecheck

# refresh static JSON data (run after ingest)
pnpm --filter @prj--personal-portfolio--v3/shared--quiz-export start
```

Dev server: http://localhost:5180

## Data

Quiz content is served from `public/data/` (generated by `shared--quiz-export`):

- `posts.json` — study-set index
- `tags.json` — tag index
- `questions/<post_slug>.json` — per-post questions
- `tags/<tag_slug>.json` — per-tag questions
- `_all.json` — full offline bundle
